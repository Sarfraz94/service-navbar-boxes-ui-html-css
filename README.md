# Service Navbar UI

A modern navigation bar interface built using HTML5 and CSS3. This project demonstrates how to create a structured website navbar using box-based layouts, inline-block elements, and CSS styling techniques.

## Overview

The navigation bar is designed for a home services website and includes a logo section, navigation menu, and search functionality. The project focuses on layout organization, alignment, spacing, and user interface design using only HTML and CSS.

## Features

* Logo Section
* Navigation Menu
* Search Bar
* Gradient Background
* Inline-Block Layout Structure
* Hover Effects on Navigation Links
* Rounded Corners
* Box Shadow Effects
* Clean and Organized Design
